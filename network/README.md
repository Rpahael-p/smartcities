Le code se synchronise sur les serveurs NTP.
Il gère les fuseaux horaires de +5 à -5.
Une pression sur le bouton permet de passer de 0 à +5, puis de -1 à -5, avant de revenir à 0.
Une double pression sur le bouton permet de changer l’affichage de 12 à 24 heures.

Le code nécessite une bonne connexion Internet, car il interroge fréquemment les serveurs NTP et attend leur réponse.
Si celle-ci tarde à arriver, le code risque de ralentir et de rendre le fonctionnement du bouton peu fiable.
